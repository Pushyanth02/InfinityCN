## 🚀 Vision

InfinityCN transforms raw novels into immersive, cinematic experiences. It prioritizes the narrative flow, emotional impact, and technical stability of the storytelling engine. This is not a reader app; it is a cinematic storytelling engine designed to wows users with its pacing and immersion.

## 🧠 Core Values

1. **Narrative First:** All decisions must improve: pacing, emotional flow, readability, and immersion.
2. **Systems > Features:** Build reusable pipelines, modular engines, and structured outputs over temporary features.
3. **No Monoliths:** Strictly follow the pipeline: `Input -> Processing -> Structured Data -> Runtime -> UI`.
4. **Cinematified + Original:** Support both the transformed cinematic version and the original text version.
5. **UI Disappears:** The interface should vanish, leaving only the story with a max typography width of ~720px.

## 🎯 Current Focus

- Expand reader discovery quality across novel, manga, manhwa, and manhua sources.
- Keep paragraph reconstruction deterministic via strategy APIs and canonical-content safeguards.
- Surface cinematic depth metrics in the reader without leaking business logic into components.
