# InfinityCN Wireframes & Key Screen Prototypes

## 1. Home Screen

```
+----------------------------------------------------------+
|  Logo      [Upload Book]   [Recent Books]   [Settings]   |
+----------------------------------------------------------+
|                                                          |
|   [ Drop file here / Click to upload ]                   |
|                                                          |
|   ┌───────────────┐   ┌───────────────┐   ┌────────────┐  |
|   |  Recent Book  |   |  Recent Book  |   |  ...      |  |
|   └───────────────┘   └───────────────┘   └────────────┘  |
|                                                          |
|   [Get Started] [How it works]                           |
+----------------------------------------------------------+
```

- Primary actions: Upload, view recent books, access settings.
- Call-to-action for new users.

## 2. Reader Screen

```
+----------------------------------------------------------+
| [Back]  Book Title         [Chapter Nav]      [Settings] |
+----------------------------------------------------------+
|                                                          |
|   [Chapter Header]                                        |
|   ----------------------------------------------------   |
|   | Cinematic Content Area (blocks, dialogue, etc.)   |   |
|   ----------------------------------------------------   |
|   [Original Text Toggle]                                 |
|                                                          |
|   [Progress Bar]                                         |
|                                                          |
|   [Footer: Prev/Next Chapter | Immersion | Preferences]   |
+----------------------------------------------------------+
```

- Focus on immersive reading, easy navigation, and quick access to settings/preferences.

---

## 3. Settings Modal

```
+------------------- Settings -----------------------------+
| [x] Close                                             |
|--------------------------------------------------------|
|  [Provider Section: cards/grid]                        |
|  [API Key Input]                                      |
|  [Preferences Section: font, size, spacing, theme]     |
|  [Test Connection]                                    |
|--------------------------------------------------------|
|  [Done]                                               |
+--------------------------------------------------------+
```

- Modular sections for provider, API keys, and preferences.
- Accessible, keyboard-friendly, and mobile-responsive layout.

---

## User Flows & Key Interactions
- Upload → Reader → Settings/Preferences → Continue Reading
- Quick switch between providers and preferences in modal
- Responsive design for mobile and desktop
